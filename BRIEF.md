# 北海道包车官网 - 建站任务 Brief

## 项目概述
为一家北海道包车公司创建官方网站。日本大公司风格：信息密度高但秩序强、信任优先、转化点分散布局。
使用占位信息（placeholder），后续替换真实内容。

## 技术栈
- **Next.js 15** (App Router) + **TypeScript** + **Tailwind CSS 4**
- 纯静态展示站，无后端/数据库
- 支持 Vercel 一键部署（`next export` 或 SSG）
- 响应式设计（移动端优先）

## 网站语言
- 默认日文界面，但所有占位文案用**中文**标注方便客户理解和替换
- 导航栏有中/日/英切换按钮（占位，不需要真正的 i18n 实现）

## 设计风格
- **日本大公司风格**：正式、清爽、可靠
- 配色：深蓝(#1B3A5C) + 白 + 浅金色点缀(#C4A35A)
- 字体：Noto Sans JP / Inter
- 大量使用北海道风景占位图（用 unsplash/placeholder）
- 信息密度高，但分组清晰，用卡片/区块分隔
- 每个关键区块都有 CTA 按钮

## 页面结构

### 1. 首页 (/)
- **Hero Section**: 全屏北海道风景背景，公司名 + Slogan + 2个CTA（立即询价、查看车型）
- **快捷服务入口**: 6张卡片（机场接送、市内包车、多日定制、滑雪接送、商务VIP、摄影亲子包车）
- **为什么选择我们**: 6个优势图标+文字（多语言司机、本地经验、透明报价、合法资质、定制行程、快速响应）
- **车型概览**: 4种车型卡片预览（带"查看详情"链接）
- **热门线路**: 横向滚动卡片，4-6条线路
- **预约流程**: 4步流程图（提交需求→顾问确认→报价方案→出行服务）
- **客户评价**: 3条精选评价卡片
- **合作伙伴Logo墙**: 占位logo
- **CTA横幅**: "开始规划您的北海道之旅" + 询价按钮
- **页脚**: 公司信息、导航链接、联系方式、社交媒体、资质信息

### 2. 服务总览 (/services)
- 按场景分类的服务详情
- 每个服务：标题、描述、适合人群、价格起点、可用车型、CTA

### 3. 机场接送 (/services/airport)
- 新千岁/旭川机场接送专页
- 接机流程、价格表、FAQ

### 4. 北海道观光包车 (/services/sightseeing)
- 一日游/多日游
- 季节推荐、人气线路

### 5. 商务VIP (/services/vip)
- 企业接待、会议用车
- 案例展示

### 6. 车型与价格 (/fleet)
- 每种车型详情卡：照片、座位数、行李、配置、起价
- 价格对照表
- 附加费用说明

### 7. 热门线路 (/routes)
- 线路列表页
- 每条线路卡片：名称、时长、季节标签、参考价

### 8. 线路详情 (/routes/[slug])
- 至少做3条示例线路详情页
- 行程安排、景点介绍、地图占位、价格、注意事项

### 9. FAQ (/faq)
- 分类折叠式FAQ
- 覆盖：费用、预约、取消、语言、儿童座椅、行李、支付

### 10. 公司介绍 (/about)
- 公司沿革、理念、团队、资质、服务区域

### 11. 客户评价 (/reviews)
- 游客评价 + 企业案例

### 12. 联系我们 (/contact)
- 联系表单（姓名、邮箱、电话、出行日期、人数、需求描述）
- 联系方式汇总（电话、邮箱、LINE、WhatsApp、微信）
- 地图占位

### 13. 博客/攻略 (/blog)
- 列表页 + 2-3篇示例文章
- SEO友好

## 全局组件
- **导航栏**: Logo + 服务/车型/线路/FAQ/关于/联系 + 语言切换 + 电话号码 + 询价按钮
- **页脚**: 完整站点地图 + 公司信息 + 资质 + 社交媒体
- **浮动CTA**: 移动端底部固定"立即询价"按钮
- **返回顶部**: 滚动出现
